Packing station using printers and labels

Packing station using printer and labels.

Here is what the packing station looks like. Notice that there's four labels on the screen. I typically set the packing stations to the ones that are being used at that exact moment or day so there's less human error. For example, if one packing station only packed one type of product, that's the only product label option it would show. I programmed it to use maximum and minimum weight limits as well, and specific preset tares, which all reduce user errors. It prompts you on the screen what went wrong. For example, you can't print off a label for a 10 lb container if you are using a 30 lb container to pack product because that's too heavy, or you can't overpack it because of weight limits. I set preset tares for the different products and containers, improving production flow, so you won't have to constantly manually tare things if you are packing many different products at once.

Weight indicators - connects to scale

Weight indicators - connects to scale

Programmning the weight indicators. I wired up different weight indicators and scales and programmed them, some without any computer 'screens' like the ones shown above earlier for the packing stations. I had to calibrate the weights and the inspections passed by scale technician that works with Measurement Canada. I was actually offered a job for this from the scale technician. Setting up scales to weight indicators means you need to open them and wire them properly, and then program it correctly to work and if you want a screen or some sort of monitor connected besides the little LED screen the weight indicator usually needs to support that via hardware and circuit board. This one can connect via ethernet. The printer for labels is technicially optional depending on what you are weighing. Weight indicators handle logic for the scales and scales (load cells) converts force to electrical signals.

Setting up processing grader
